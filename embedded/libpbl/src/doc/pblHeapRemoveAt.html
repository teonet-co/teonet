void* pblHeapRemoveAt

( PblHeap* heap, int index )

Removes the element at the specified position from the heap, maintaining the heap condition of the heap.

Documentation

Removes the element at the specified position from the heap, maintaining the heap condition of the heap.

This function has a time complexity of O(Log N), with N being the number of elements in the heap.

Parameters:
heap - The heap to use
index - The index at which the element is to be removed
Returns:
void* retptr != (void*)-1: The element removed.
void* retptr == (void*)-1: An error see pbl_errno:
PBL_ERROR_OUT_OF_BOUNDS - Index is out of range (index < 0 || index >= size()).
