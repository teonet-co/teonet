void pblHeapFree

( PblHeap* heap )

Frees the heap's memory from heap.

Documentation

Frees the heap's memory from heap.

Note: The memory of the elements themselves is not freed.

This function has a time complexity of O(N), with N being the number of elements in the heap.

Parameters:
heap - The heap to free
Returns:
void
