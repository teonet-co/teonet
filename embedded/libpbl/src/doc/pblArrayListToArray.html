void* pblArrayListToArray

( pblArrayList* arrayList )

Returns an array containing all of the elements in this list in the correct order.

Documentation

Returns an array containing all of the elements in this list in the correct order.

The pointer array returned is malloced from heap, the caller has to free it after it is no longer needed!

The size of the pointer array malloced and returned is defined by the size() of the list.

Parameters:
arrayList - The list to use
Returns:
void * retptr != NULL: The array containing the elements of the list.
void * retptr == NULL: An error, see pbl_errno PBL_ERROR_OUT_OF_MEMORY - out of memory
