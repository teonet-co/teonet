static int pblTreeSetContains

( PblTreeSet* set,
  void* element )

Returns true if this tree set contains the specified element.

Documentation

Returns true if this tree set contains the specified element.

This method has a time complexity of O(Log N), with N being the size of the set.

Parameters:
set - The set to use
element - Element to look for
Returns:
int rc != 0: The specified element is present.
int rc == 0: The specified element is not present.
