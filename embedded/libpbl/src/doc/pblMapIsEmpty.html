int pblMapIsEmpty

( PblMap* map )

Tests if this map has no elements.

Documentation

Tests if this map has no elements.

This method has a time complexity of O(1).

Parameters:
map - The map to test
Returns:
int rc != 0: This map has no elements.
int rc == 0: This map has elements.
