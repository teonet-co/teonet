void pblSetClear

( PblSet* set )

Removes all of the elements from this set.

Documentation

Removes all of the elements from this set.

Note: No memory of the elements themselves is freed.

This method has a time complexity of O(N), with N being the size of the set.

Parameters:
set - The set to clear
Returns:
void
