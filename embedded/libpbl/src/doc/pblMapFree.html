void pblMapFree

( PblMap* map )

Removes all of the mappings from this map and frees the map's memory from heap.

Documentation

Removes all of the mappings from this map and frees the map's memory from heap.

For hash maps this method has a time complexity of O(N). For tree maps this method has a time complexity of O(N * Log N).

Parameters:
map - The map to free
Returns:
void
