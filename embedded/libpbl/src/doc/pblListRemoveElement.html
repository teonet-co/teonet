int pblListRemoveElement

( PblList* list, void* element )

Removes a single instance of the specified element from this list, if it is present.

Documentation

Removes a single instance of the specified element from this list, if it is present.

More formally, removes an element e such that (o==null ? e==null : o.equals(e)), if the list contains one or more such elements. Returns true if the list contained the specified element (or equivalently, if the list changed as a result of the call).

This method has a time complexity of O(N), with N being the size of the list.

Parameters:
list - The list to use
element - The element to remove
Returns:
int rc != 0: The list contained the specified element.
int rc == 0: The specified element is not present.
