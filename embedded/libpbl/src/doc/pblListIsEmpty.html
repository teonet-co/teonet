int pblListIsEmpty

( PblList* list )

Tests if this list has no elements.

Documentation

Tests if this list has no elements.

This method has a time complexity of O(1).

Parameters:
list - The list to test
Returns:
int rc != 0: This list has no elements.
int rc == 0: This list has elements.
